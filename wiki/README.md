# Wiki sources

Ten katalog (`wiki/`) zawiera źródła stron GitHub Wiki (Markdown).

Publikacja:

```bash
./scripts/publish_wiki.sh
```

Uwagi:

- GitHub Wiki to osobne repo: `<repo>.wiki.git`
- linki do plików w repo używają placeholderów `{{REPO_URL}}` i `{{DEFAULT_BRANCH}}` (podmienia je skrypt)
